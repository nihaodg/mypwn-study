寄存器基础

寄存器是CPU中最快的存储单元。64位系统主要有以下寄存器：

通用寄存器
RAX, RBX, RCX, RDX
R8-R15 (64位新增)
指针寄存器
RSP (栈顶)
RBP (栈底)
指令指针
RIP
指向下一条指令
标志寄存器
RFLAGS
ZF, SF, OF等
64位寄存器向后兼容
RAX → EAX → AX → AH/AL (层层拆分)

内存模型

高地址 ← Stack (栈) 自动管理，向低地址增长
Heap (堆) 动态分配，malloc/free
.data 已初始化全局变量
.bss 未初始化全局变量
.text 代码段(指令)
低地址 → Kernel

栈的基本概念

LIFO - 后进先出
栈就像叠盘子，最后放上去的盘子最先被拿走。
push - 入栈
push rax → RSP -= 8, [RSP] = RAX
pop - 出栈
pop rax → RAX = [RSP], RSP += 8
call - 函数调用
push RIP(下一条指令地址), jmp 函数
ret - 函数返回
pop RIP → 返回调用点继续执行

数据传送指令

mov - 赋值
mov rax, 10 ; rax = 10
mov rax, rbx ; rax = rbx
mov [rsi], rax ; [rsi] = rax (内存操作)
lea - 取地址
lea rax, [rbx+8] ; rax = rbx + 8 (不访存)
xchg - 交换
xchg rax, rbx ; 交换两个寄存器的值
