汇编指令详解

掌握x86/x64核心指令集，理解每条指令的执行效果。

0基础

数据传送

MOV 目标, 源
mov rax, 5 ; rax = 5
mov rax, rbx ; rax = rbx
mov [rsi], rdi ; 内存操作
注意: 不能同时为内存操作
LEA 取地址
lea rax, [rbx+8] ; rax = rbx+8
lea rax, [rdi+rsi*4]; rax = rdi+4*rsi
常用于计算地址，不访问内存
XCHG 交换
xchg rax, rbx ; 交换值

算术运算

ADD SUB
add rax, 5 ; rax = rax + 5
sub rax, rbx ; rax = rax - rbx
影响ZF, SF, OF, CF等标志位
IMUL IDIV
imul rax, rbx ; rdx:rax = rax*rbx
idiv rcx ; rdx:rax / rcx
有符号乘除法
INC DEC
inc rax ; rax++
dec rax ; rax--
不影响CF
NEG
neg rax ; rax = -rax

逻辑运算

AND OR XOR NOT
and rax, rbx ; 按位与
or rax, 0xFF ; 按位或
xor rax, rax ; 常用: 清零
not rax ; 按位取反
xor rax, rax 是常用的清零方式，比 mov rax,0 更快
SHL SHR SAL SAR
shl rax, 1 ; 左移1位 (*2)
shr rax, 2 ; 右移2位 (/4)
sar rax, 1 ; 算术右移(保留符号)
TEST CMP
test rax, rax ; rax & rax (设置ZF)
cmp rax, 5 ; rax - 5 (只设置标志位)
CMP通过计算差值设置标志位，不保存结果

控制流

JMP 无条件跳转
jmp label ; 跳转到label
jmp rax ; 跳转到rax存的地址
jmp [rax] ; 间接跳转
JE JNE JL JLE JG JGE
je label ; ZF=1 时跳转 (=)
jne label ; ZF=0 时跳转 (!=)
jl label ; SF≠OF 时跳转 (<)
jle label ; ZF=1 or SF≠OF (<=)
有符号比较
JA JB JAE
ja label ; CF=0 and ZF=0 (无符号>)
jb label ; CF=1 (无符号<)
jae label ; CF=0 (无符号>=)
无符号比较
LOOP CALL RET
loop label ; rcx--, if rcx!=0: jmp
call func ; push RIP, jmp func
ret ; pop RIP

标志寄存器 (RFLAGS)

CF
进位标志
PF
奇偶标志
AF
辅助进位
ZF
零标志
SF
符号标志
TF
陷阱标志
IF
中断标志
OF
溢出标志
ZF (Zero Flag)
结果为0时置1，否则为0
常用: test rax, rax 后 je 判断是否为零
OF (Overflow Flag)
有符号运算溢出时置1
常用: jo 检测溢出
