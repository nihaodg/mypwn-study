汇编实战分析

通过真实代码示例，学习如何阅读和理解汇编代码。

实战

示例1: 简单加法函数

C 源代码 
long add(long a, long b) {
    return a + b;
}
x64 汇编 
add:
    lea   rax, [rdi + rsi]    ; rax = rdi + rsi
    ret                     ; 返回
使用 lea 而非 add，可以在一行内完成加法和返回
分析
  • 参数 a 在 RDI 中（x64 Linux约定）
  • 参数 b 在 RSI 中
  • 返回值在 RAX 中
  • 使用 lea 可以直接计算并赋值，无需修改寄存器

示例2: 带局部变量的函数

C 源代码 
long calc(long x, long y) {
    long a = x * 2;     // 局部变量a
    long b = y + 10;    // 局部变量b
    return a + b;       // 返回 a + b
}
x64 汇编 
calc:
    push  rbp
    mov   rbp, rsp
    sub   rsp, 0x10        ; 分配 16 字节栈空间
    
    mov   [rbp-8], rdi      ; a = x * 2
    add   [rbp-8], rdi      ; a += x
    
    mov   [rbp-16], rsi     ; b = y + 10  
    add   [rbp-16], 10      ; b += 10
    
    mov   rax, [rbp-8]      ; rax = a
    add   rax, [rbp-16]     ; rax += b
    
    leave
    ret
栈布局
RBP-8 → 局部变量 a
RBP-16 → 局部变量 b
RBP → 保存的 RBP
RBP+8 → 返回地址
关键指令
  • sub rsp, 0x10 - 分配栈空间
  • mov [rbp-8], rdi - 存储局部变量
  • leave = mov rsp, rbp; pop rbp

示例3: 条件分支

C 源代码 
int abs(int x) {
    if (x >= 0)
        return x;
    else
        return -x;
}
x64 汇编 
abs:
    mov   eax, edi          ; eax = x (EDI低32位)
    test  eax, eax         ; eax & eax (设标志位)
    jns   .L_return        ; if SF=0 (>=0) 跳转
    
    neg   eax               ; eax = -eax
    
.L_return:
    ret                     ; 返回
关键指令分析
test eax, eax
通过AND运算设置标志位，不改变值
jns (Jump if Not Signed)
SF=0时跳转，即x>=0时跳转到返回

示例4: 循环

C 源代码 
long sum(long n) {
    long total = 0;
    for (long i = 1; i <= n; i++)
        total += i;
    return total;
}
// sum(5) = 1+2+3+4+5 = 15
x64 汇编 
sum:
    push  rbp
    mov   rbp, rsp
    sub   rsp, 0x10
        
    mov   [rbp-8], 0        ; total = 0
    mov   [rbp-16], 1        ; i = 1
    
.loop:
    mov   rax, [rbp-16]     ; rax = i
    cmp   rax, rdi          ; i - n
    jg    .done             ; if i > n 跳出循环
    
    add   [rbp-8], rax      ; total += i
    add   [rbp-16], 1        ; i++
    jmp   .loop             ; 跳转回循环
    
.done:
    mov   rax, [rbp-8]      ; 返回 total
    leave
    ret
